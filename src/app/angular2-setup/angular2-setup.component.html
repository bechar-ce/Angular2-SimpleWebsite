Set up / Install Node JS Environment

  • Download the Node.js installable archive file from Node.js web site.
  • Please choose the installabe archive file as per your operating system as below
  • Windows: node-v6.x-x64.msi
  • Linux: node-v6.x-linux-x86.tar.gz
  • Mac OS: node-v6.x-darwin-x86.tar.gz
  • SunOS: node-v6.x-sunos-x86.tar.gz

After successfully installation of the Node.js you need to install the angular-cli. It's so easy to setup the angular-cli. Please find the below sets for the installation of the angular-cli on your local environment. The Angular 2 CLI makes it easy to create an application that just works out of the box.

Set up / Install Node JS Environment

  • The Angular CLI makes it so easy and simple to create an angular 2 application with few steps and configuration speed
  • Please check the information about the Angular CLI from this link
  • Run the command on command prompt "npm install -g @angular/cli"
  • To update Updating Angular CLI please follow below commands
  • npm uninstall -g angular-cli
  • npm uninstall --save-dev angular-cli
  • For the Global package:
  • npm uninstall -g @angular/cli
  • npm cache clean
  • npm install -g @angular/cli@latest

Create Hello World Project from Angular CLI

  • Creating project with Angular CLI is so simple just put this line on command window ng new < PROJECT_NAME >
  • If you seen in the left side image it's create the project structure after hitting the command on command line i.e. ng new hello-world
  • You can run your application with help of command ng serve
  • It can navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files in the editor.
  • You can configure the default HTTP host and port used by the development server with simple two command-line options ng serve --host 0.0.0.0 --port 4201
  • To more informaiton you can navigate to this Angular CLI Know More
